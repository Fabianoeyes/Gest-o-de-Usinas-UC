# Gest-o-de-Usinas-UC

Aplicação em Streamlit para acompanhar a planilha de gestão de usinas.

## Como usar

1. Coloque o arquivo Excel na mesma pasta do `app.py` (o padrão esperado é `Gestao_de_Usinas_e_UCs.xlsx`).
2. Caso o nome padrão não exista, o aplicativo procura automaticamente o arquivo `.xlsx` mais recente no diretório e informa qual foi carregado na barra lateral.
3. Execute a aplicação com `streamlit run app.py`.
